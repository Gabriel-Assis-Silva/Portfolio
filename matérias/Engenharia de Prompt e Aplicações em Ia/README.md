# 🤖 Engenharia de Prompt e Aplicações em IA

## 📝 Descrição do Projeto
Esta disciplina aborda as técnicas e estratégias para interagir eficientemente com modelos de linguagem de grande escala (LLMs), como o ChatGPT e o GitHub Copilot. O foco está em desenvolver habilidades de engenharia de prompt para extrair resultados precisos e úteis de ferramentas de IA, bem como compreender suas aplicações práticas no desenvolvimento de software e na resolução de problemas.

Os projetos desenvolvidos contemplam a criação de prompts estruturados, aplicação de técnicas como few-shot learning e chain-of-thought, além da exploração de ferramentas de IA generativa para produtividade e criação de conteúdo.

## 🚀 Tecnologias Utilizadas
* **Modelos de IA:** ChatGPT (OpenAI), GitHub Copilot
* **Ferramentas:** Google Colab, VS Code
* **Técnicas:** Zero-shot, Few-shot, Chain-of-Thought Prompting

## 📊 Resultados e Aprendizados
* **Engenharia de Prompt:** Aprendi a estruturar prompts claros e eficientes para obter respostas precisas de modelos de IA.
* **Automação de Tarefas:** Explorei como usar IA para automatizar tarefas repetitivas de desenvolvimento, como geração de código, documentação e testes.
* **Análise Crítica de Saídas:** Desenvolvi habilidade para avaliar criticamente as respostas geradas por IA, identificando possíveis erros ou vieses.
* **Aplicações Práticas:** Pratiquei o uso de IA para resolução de problemas reais de programação e criação de conteúdo.

## 🔧 Como Navegar
1. Acesse as pastas do projeto para visualizar os exemplos de prompts e suas respostas.
2. Os notebooks `.ipynb` contêm experimentos práticos com modelos de IA e análise dos resultados.
