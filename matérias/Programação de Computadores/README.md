# ⚙️ Programação de Computadores

## 📝 Descrição do Projeto
Esta disciplina introduz os fundamentos da programação estruturada utilizando a linguagem C. Os projetos desenvolvidos cobrem desde conceitos básicos como variáveis, operadores e estruturas de controle, até tópicos mais avançados como ponteiros, alocação dinâmica de memória, structs e manipulação de arquivos.

O aprendizado é aplicado na resolução de problemas computacionais clássicos, desenvolvendo o raciocínio lógico e a capacidade de construir algoritmos eficientes.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** C (padrão C99/C11)
* **Compilador:** GCC
* **Ferramentas:** VS Code, Code::Blocks

### Principais Aprendizados
* **Ponteiros e Memória:** Aprendi a gerenciar memória manualmente com `malloc`, `calloc` e `free`, compreendendo o funcionamento da memória em nível baixo.
* **Estruturas de Dados:** Implementei estruturas como listas encadeadas, filas e pilhas em C puro.
* **Algoritmos:** Pratiquei algoritmos de ordenação (Bubble Sort, Selection Sort) e busca (linear e binária).
* **Depuração:** Desenvolvi habilidade para identificar e corrigir erros de segmentação e vazamentos de memória.

## 🔧 Como Executar
1. Instale o GCC: `sudo apt install gcc` (Linux) ou MinGW (Windows).
2. Compile o projeto: `gcc -o programa main.c`.
3. Execute: `./programa` (Linux/Mac) ou `programa.exe` (Windows).
