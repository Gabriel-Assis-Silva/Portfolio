# 🎨 Interface e Jornada de Usuário

## 📝 Descrição do Projeto
Esta disciplina explora os fundamentos de UX (User Experience) e UI (User Interface), com foco em compreender as necessidades dos usuários e projetar experiências digitais centradas neles. Os projetos desenvolvidos envolvem pesquisa com usuários, criação de personas, mapeamento de jornadas e design de interfaces acessíveis e intuitivas.

O aprendizado prático inclui técnicas como entrevistas com usuários, análise de heurísticas de usabilidade, criação de fluxos de navegação e testes de usabilidade.

## 🚀 Tecnologias Utilizadas
* **Design:** Figma
* **Metodologias:** Design Centrado no Usuário (DCU), Pesquisa Qualitativa
* **Entregas:** Personas, Mapas de Jornada, Wireframes, Protótipos

## 📊 Resultados e Aprendizados
* **Criação de Personas:** Aprendi a construir perfis detalhados de usuários baseados em dados reais de pesquisa.
* **Mapeamento de Jornada:** Desenvolvi habilidade para identificar pontos de dor e oportunidades de melhoria na experiência do usuário.
* **Heurísticas de Nielsen:** Pratiquei a avaliação de interfaces com base nos 10 princípios de usabilidade de Jakob Nielsen.
* **Design Acessível:** Aprendi a aplicar diretrizes de acessibilidade (WCAG) no design de interfaces.

## 🔧 Como Navegar
1. Acesse as pastas do projeto para visualizar os documentos de pesquisa e artefatos de design.
2. Os arquivos do Figma contêm protótipos navegáveis com fluxos de usuário completos.
