# 🖥️ Prototipagem de Sistemas Computacionais

## 📝 Descrição do Projeto
Esta disciplina aborda o processo de criação de protótipos para sistemas computacionais, desde a concepção inicial até a validação de interfaces com usuários reais. O objetivo é desenvolver habilidades de design de sistemas, utilizando metodologias ágeis e ferramentas de prototipagem para transformar requisitos em interfaces funcionais e intuitivas.

Os projetos desenvolvidos contemplam a criação de wireframes, protótipos de baixa e alta fidelidade, bem como a aplicação de princípios de usabilidade e acessibilidade no design de sistemas.

## 🚀 Tecnologias Utilizadas
* **Ferramentas de Prototipagem:** Figma, Canva
* **Metodologias:** Design Thinking, Prototipagem Rápida
* **Documentação:** Markdown, PDF

## 📊 Resultados e Aprendizados
* **Design de Interfaces:** Aprendi a criar protótipos navegáveis que simulam o fluxo real de um sistema antes do desenvolvimento.
* **Validação com Usuários:** Desenvolvi habilidade para coletar feedback e iterar rapidamente sobre protótipos.
* **Especificação de Requisitos:** Pratiquei a tradução de requisitos funcionais em telas e fluxos de navegação claros.

## 🔧 Como Navegar
1. Acesse as pastas do projeto para visualizar os arquivos de documentação e protótipos.
2. Abra os arquivos `.fig` no Figma ou os `.pdf` de apresentação para visualizar os protótipos.
